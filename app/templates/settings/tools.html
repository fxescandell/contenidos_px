9. Herramientas y Test

Verifica que todos los servicios y conexiones del sistema funcionan correctamente. Puedes ejecutar los tests de forma individual o lanzar todos a la vez.

Pulsa un boton para ejecutar los tests
🗃 Base de datos
Conexion a la DB, consulta SELECT 1, tablas de modelos
⚙ Tabla de configuracion
Acceso a system_settings y SettingsResolver
📁 Rutas locales
Verifica que working, export, temp y logs existen y son writables
💻 Sistema
CPU, memoria, disco, version de Python
🏠 SMB / Hotfolder
Conexion SMB con Synology, listado de shares y carpetas
📤 FTP / Outfolder
Conexion FTP de salida, escritura de test
✉ Telegram
Envia un mensaje de prueba al bot configurado
🤖 LLM activo
Prueba la conexion con el proveedor LLM activo (OpenAI, Anthropic, Groq...)
✎ WordPress API
Conexion REST API de WordPress con auth
🔄 Recarga de cache
Fuerza la recarga de la cache de configuracion desde la DB
⚡ Test de pipeline (lote completo)
Crea ficheros de prueba, los pasa por todo el pipeline (ingestion, clasificacion, exportacion) y muestra los resultados. Incluye opciones de municipio y categoria.
🛠 Test por flujo
Ejecuta un flujo existente (ingestion + clasificacion + JSON). Selecciona un flujo configurado en la seccion de Flujos.
